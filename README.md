# Descrição do Projeto

Aplicação Console em C# desenvolvida para fins de estudo, aplicando conceitos de Programação Orientada a Objetos como herança, classes abstratas, interfaces e o padrão Repository. O projeto inclui uma máquina de estados para controle do fluxo da aplicação e manipulação de listas, permitindo adicionar, remover e listar elementos, com foco em organização, clareza e boas práticas de código.

O projeto tem caráter educacional, servindo como base para aprendizado, experimentação e consulta futura sobre conceitos essenciais da linguagem C# e da orientação a objetos.
